Real-Time Product Analytics with Kafka Streams

This project demonstrates how to build a real-time product analytics platform using Kafka Streams, solving common challenges in e-commerce analytics at scale.

Features

Real-time Processing

  • Product event processing
  • Time-windowed analytics
  • Unique user tracking

Analytics

  • Aggregated product statistics
  • State store management
  • Scalable stream processing

Operations

  • Fault-tolerant operations
  • Monitoring & metrics
  • Easy deployment

Prerequisites

  • Java 17+
  • Docker and Docker Compose
  • Gradle

Quick Start

# Start Kafka environment
docker-compose up -d

# Load sample data
./gradlew loadProductMetadata

# Run the application
./gradlew bootRun

For detailed setup instructions, see our Getting Started Guide.
